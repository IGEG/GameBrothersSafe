# GameBrothersSafe
Игра построена на WPF (C#).

Игра состоит из 3 уровенй сложности:
- Легкий (массив 3 на 3) EasyGame
- Средний (массив 4 на 4) AverageGame
- Сложный (массив 5 на 5) DifficultGame

Вся логика игры инкапсулируется в классах ViewModel, реализующие интерфейс IGameModel (Services/IGameModel (общий принцип игры)).
